## Building from Source

### Windows
```bash
git clone https://github.com/dylmeadows/EonTimer && cd EonTimer
gradlew.bat distZip
```

### MacOS / Linux
```bash
git clone https://github.com/dylmeadows/EonTimer && cd EonTimer
./gradlew distZip
```
